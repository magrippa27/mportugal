## Cabeçalhos

Os seguintes elementos HTML `<h1>`—`<h6>` representam seis níveis de cabeçalhos de secção. `<h1>` é o nível de secção mais alto enquanto `<h6>` é o mais baixo.

# H1

## H2

### H3

#### H4

##### H5

###### H6

## Citações

O elemento blockquote representa conteúdo que é citado de outra fonte.

#### Sintaxe

```markdown
> Tiam, ad mint andaepu dandae nostion secatur sequo quae.  
> **Nota** que pode usar _sintaxe Markdown_ dentro de uma citação.
```

#### Resultado

> Tiam, ad mint andaepu dandae nostion secatur sequo quae.\
> **Nota** que pode usar *sintaxe Markdown* dentro de uma citação.

## Tabelas

#### Sintaxe

```markdown
| Itálico   | Negrito  | Código |
| --------- | -------- | ------ |
| _itálico_ | **negrito** | `código` |
```

#### Resultado

| Itálico   | Negrito  | Código |
| --------- | -------- | ------ |
| *itálico* | **negrito** | `código` |

## Blocos de Código

#### Sintaxe

Podemos usar 3 acentos graves \`\`\` numa nova linha e escrever código e fechar com 3 acentos graves numa nova linha e para destacar sintaxe específica de linguagem, escrever uma palavra do nome da linguagem após os primeiros 3 acentos graves, por exemplo html, javascript, css, markdown, typescript, txt, bash

````markdown
```html
<!doctype html>
<html lang="pt">
  <head>
    <meta charset="utf-8" />
    <title>Exemplo de Documento HTML5</title>
  </head>
  <body>
    <p>Teste</p>
  </body>
</html>
```
````

#### Resultado

```html
<!doctype html>
<html lang="pt">
  <head>
    <meta charset="utf-8" />
    <title>Exemplo de Documento HTML5</title>
  </head>
  <body>
    <p>Teste</p>
  </body>
</html>
```

## Tipos de Lista

### Lista Ordenada

#### Sintaxe

```markdown
1. Primeiro item
2. Segundo item
3. Terceiro item
```

#### Resultado

1. Primeiro item
2. Segundo item
3. Terceiro item

### Lista Não Ordenada

#### Sintaxe

```markdown
- Item de lista
- Outro item
- E outro item
```

#### Resultado

* Item de lista
* Outro item
* E outro item

### Lista Aninhada

#### Sintaxe

```markdown
- Fruta
  - Maçã
  - Laranja
  - Banana
- Lacticínios
  - Leite
  - Queijo
```

#### Resultado

* Fruta
  * Maçã
  * Laranja
  * Banana
* Lacticínios
  * Leite
  * Queijo
