Este tema vem com a integração [@astrojs/mdx](https://docs.astro.build/pt/guides/integrations-guide/mdx/) instalada e configurada no seu ficheiro de configuração `astro.config.mjs`. Se preferir não usar MDX, pode desativar o suporte removendo a integração do seu ficheiro de configuração.

## Porquê MDX?!

MDX é um sabor especial de Markdown que suporta sintaxe JavaScript e JSX incorporada. Isto permite [misturar JavaScript e Componentes UI no seu conteúdo Markdown](https://docs.astro.build/pt/guides/markdown-content/#mdx-features) para coisas como gráficos interativos ou alertas.

Se tiver conteúdo existente escrito em MDX, esta integração tornará a migração para Astro mais fácil.
